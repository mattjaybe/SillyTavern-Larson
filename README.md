## 🚀 Installation

<details open>
<summary id="installation"><b>📦 Click to see installation steps</b></summary>

<br>

1. Open **SillyTavern**
2. Click the **Extensions** icon (puzzle piece)
3. Click **Install Extension**
4. Paste this URL:
   ```
   https://github.com/mattjaybe/SillyTavern-Larson
   ```
5. Click **Install** and refresh

**That's it!** The animated bar will appear above your message input.

</details>

## 🎮 Quick Start Guide

<details open>
<summary id="quick"><b>⚡ Get started in 30 seconds</b></summary>

<br>

### Basic Setup
1. **Click the animated bar** to open settings
2. Choose your **Animation Style** (try Gradient or Cylon!)
3. Select a **Theme** from the dropdown
4. Pick an **Animation Speed** (Normal is great for most)

### Enable Optional States
- Toggle **Idle** for animation when waiting
- Toggle **Thinking** for reasoning phases (requires Streaming)
- Enable **Hide When Idle** for a cleaner look

### Create a Custom Theme
1. Scroll to **Custom Themes** → **Add custom theme**
2. Name your theme
3. Adjust 4 color sliders (Background, Primary, Secondary, Accent)
4. Watch live preview → Click **Save**

💡 **Pro Tip**: Hit the Randomize button for instant color inspiration!

</details>

## 🐛 Troubleshooting

<details>
<summary><b>❓ Common issues and solutions</b></summary>

<br>

### Bar not appearing
- Ensure **Enable Larson** toggle is ON
- Check that you're in a chat (not settings page)
- Try refreshing the page
- Verify extension is in `public/scripts/extensions/third-party/`

### Thinking animation not working
- Toggle **Thinking** must be enabled
- Must use **Streaming** mode (not Instruct/Complete)
- Your LLM must output thinking tags

### Animations feel choppy
- Try a different animation style
- Reduce speed to Slow
- Close unnecessary browser tabs
- Update browser to latest version

### Custom theme not saving
- Ensure theme name is not empty
- Try a different name
- Verify localStorage is enabled

</details>
